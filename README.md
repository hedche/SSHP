# SSHP
An IoT system that can monitor, log, control, and alert using the following hardware: 
- Arduino Boards (ESP8266)
- Raspberry Pi
- Web hosting service

Languagues used:
- C ++
- SQL
- PHP
- HTML

This platform was tested using a TP-Link M7200 4G/3G mobile router.
